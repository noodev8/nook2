
Flutter Module

This project uses a modular Flutter structure. When generating screens or components, follow the conventions below.

## Folder Structure

lib/
├── main.dart
├── app.dart
├── config/
│   └── app_config.dart
│   └── theme.dart
├── navigation/
│   └── router.dart
├── screens/
│   └── <screen_name>/<screen_name>_screen.dart
├── components/
│   └── <custom_component>.dart
├── styles/
│   └── spacing.dart
└── api/
    └── <api_name>.dart

## UI Rules

- Use Flutter with Material 3
- Use `AppTheme.lightTheme` for theming
- Use `AppSpacing` for margins/paddings
- Use reusable widgets from `/components/` where appropriate
- Use `GoRouter` for navigation
- Always create new screens under `/screens/<name>/<name>_screen.dart`
- When generating a widget, make it standalone and configurable
